Si vous modifiez en LOCAL (sur votre PC avec Notepad++) :

Modifier vos fichiers
Git GUI → Rescan (pour voir les modifications)
Stage Changed (pour sélectionner les fichiers à envoyer)
Commit (avec un message qui explique ce que vous avez fait)
Push (pour envoyer sur GitHub)

Si vous modifiez directement sur GITHUB (via le site web) :

Faire vos modifications sur le site
Avant de travailler en local, faire un Pull dans Git GUI (Remote → Fetch, puis Merge → Local Merge) pour récupérer les changements
